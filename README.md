## Run the project on your machine

Clone git repository

```bash
 git clone https://github.com/FardinOA/munchies-task
```
Go to project folder

```bash
  cd munchies-task
```

Install packeges

```bash
  npm install or yarn
```

Run locally

```bash
  npm run dev or yarn dev
```
